## Kapp Form

### Props

<Props of={KappForm} />

### Examples

#### Simple implementation of a `KappForm`.

```javascript
<KappForm kappSlug="services" />
```

#### Control the Layout of a `KappForm`.

```javascript
<KappForm>
  {({ form }) => (
    <div>
      <h5>My Form Header</h5>
      {form}
    </div>
  )}
</KappForm>
```

Because no name was passed, submitting this form will create a new Kapp

