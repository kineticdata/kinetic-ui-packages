## Form Type

### Props

<Props of={FormTypeForm} />

### Examples

#### Simple implementation of a `FormTypeForm`.

```javascript
<FormTypeForm kappSlug="services" />
```

Because no name was passed, submitting this form will create a new Form

#### Control the Layout of a `FormTypeForm`.

```javascript
<FormTypeForm kappSlug="services">
  {({ form }) => (
    <div>
      <h5>My Form Header</h5>
      {form}
    </div>
  )}
</FormTypeForm>
```

Because no name was passed, submitting this form will create a new Space Attribute Definition
