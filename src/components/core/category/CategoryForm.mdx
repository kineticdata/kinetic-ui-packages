## Category Form

### Props

<Props of={CategoryForm} />

### Examples

#### Simple implementation of a `CategoryForm`.

```javascript
<CategoryForm kappSlug="services" />
```

Because no name was passed, submitting this form will create a new Category

#### Control the Layout of a `CategoryForm`.

```javascript
<CategoryForm kappSlug="services">
  {({ form }) => (
    <div>
      <h5>My Form Header</h5>
      {form}
    </div>
  )}
</CategoryForm>
```

Because no name was passed, submitting this form will create a new Space Attribute Definition
