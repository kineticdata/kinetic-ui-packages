## Bridge Model Form

### Props

<Props of={BridgeModelForm} />

### Examples

#### Simple implementation of a `BridgeModelForm`.

```javascript
<BridgeModelForm />
```

#### Control the Layout of a `KappForm`.

```javascript
<BridgeModelForm>
  {({ form }) => (
    <div>
      <h5>My Form Header</h5>
      {form}
    </div>
  )}
</BridgeModelForm>
```

Because no name was passed, submitting this form will create a new Bridge Model
