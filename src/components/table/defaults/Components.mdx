# Default Table Components

The following components are simply default components and cannot be imported.
These are examples of components that you can override.

## BodyCell

This component represents a `<td>` tag as used in the body of a table. It
can be used to override all body cells or only the cells of specific columns.

### Props

<Props of={BodyCell} />

### Default Component

```js static
const BodyCell = ({ index, row, value }) => (
  <td {...(index === 0 ? { scope: 'row' } : {})}>{value}</td>
);
```

## BodyRow

This component represents the `<tr>` tag within a table body.

### Props

<Props of={BodyRow} />

### Default Component

```js static
const BodyRow = ({ cells }) => <tr>{cells}</tr>;
```

## EmptyBodyRow

This component is rendered when there are no rows being rendered.

### Props

<Props of={EmptyBodyRow} />

### Default Component

```js static
const EmptyBodyRow = ({ colSpan }) => (
  <tr>
    <td className="text-center" colSpan={colSpan}>
      <em>
        <I18n>No data found.</I18n>
      </em>
    </td>
  </tr>
);
```

## HeaderCell

This represents the `<th>` tag in a table header row.

### Props

<Props of={HeaderCell} />

### Default Component

```js static
const HeaderCell = ({ onSortColumn, sortable, sorting, title }) => (
  <th {...(title ? { scope: 'col' } : {})}>
    {sortable === false ? (
      <I18n>{title}</I18n>
    ) : (
      <span>
        <I18n>{title}</I18n>
        <button type="button" onClick={onSortColumn}>
          {sorting === 'desc' ? (
            <span>&darr;</span>
          ) : sorting === 'asc' ? (
            <span>&uarr;</span>
          ) : (
            <span>&ndash;</span>
          )}
        </button>
      </span>
    )}
  </th>
);
```

## HeaderRow

This component represents the `<tr>` tag within a `<thead>`.

### Props

<Props of={HeaderRow} />

### Default Component

```js static
const HeaderRow = ({ columnHeaders }) => <tr>{columnHeaders}</tr>;
```

## PaginationControl

This component controls the rendering of the pagination controls.

### Props

<Props of={PaginationControl} />

### Default Component

```js static
export const PaginationControl = ({ nextPage, prevPage, loading }) => (
  <div>
    <button onClick={prevPage} disabled={loading || !prevPage}>
      <I18n>Previous</I18n>
    </button>
    <button onClick={nextPage} disabled={loading || !nextPage}>
      <I18n>Next</I18n>
    </button>
  </div>
);
```

## TableBody

This component represents the `<tbody>` tag within a `<table>`.

### Props

<Props of={TableBody} />

### Default Component

```js static
const TableBody = ({ tableRows }) => <tbody>{tableRows}</tbody>;
```

## TableLayout

This component allows you to override the `<table>` component.

### Props

<Props of={TableLayout} />

### Default Component

```js static
const TableLayout = ({ header, body, footer }) => (
  <table>
    {header}
    {body}
    {footer}
  </table>
);
```
